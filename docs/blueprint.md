# **App Name**: StudyParadox

## Core Features:

- Google Authentication: Allow students to log in securely using their Google accounts.
- Room Creation: Enable students to create video calling rooms for group study.
- Active Room Display: Display a list of active study rooms with the number of participants.
- Room Joining: Allow students to join existing study rooms to collaborate with others.
- Moderation tools: Provide basic moderation tools within each room, such as muting participants.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and knowledge.
- Background color: Very light blue (#E8EAF6), a desaturated tint of the primary.
- Accent color: Violet (#7E57C2) to provide visual interest.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and short pieces of text and 'Inter' (sans-serif) for body text.
- Use clean and modern icons to represent room status and actions.
- Design a clean and intuitive layout with a focus on easy navigation.
- Use subtle transitions and animations to enhance the user experience.
